# Users CRUD App

#### Simple CRUD application built with Vanilla JavaScript and Bootstrap.

### Live View
https://crud-users-vanillajs.netlify.app/

### Features
- View users
- Create user
- Edit user
- Delete user
- Basic form validation
- Toast notifications
- Loading spinner

### Tech Stack
- JavaScript
- Bootstrap 5

### Run Project
1. Download or clone the repo
2. Open index.html in your browser
### API

Uses: https://jsonplaceholder.typicode.com/users

### Note: Data is not saved on the server (fake API)

### Structure
- ##### Model — API & data logic
- ##### View — UI (table, modals, toasts)
- ##### Controller — app logic
